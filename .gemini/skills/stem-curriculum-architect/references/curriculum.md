# STEM Curriculum Reference (Ages 2–5)

Every game must map to one of these domains and follow the developmental ladder.

## Domains and Ladder

### Number Sense
- **2–3**: Subitize 1–3 (recognize quantity without counting).
- **3–4**: Count 1–5 with 1:1 correspondence (touching each item).
- **4–5**: Count to 10 + numeral recognition ("how many?").

### Geometry / Shapes
- **2–3**: Circle, square.
- **3–4**: + Triangle, star, heart.
- **4–5**: Match + name, tracing.

### Patterns / Algebra
- **2–3**: AB patterns (Red, Blue, Red, Blue).
- **3–4**: ABC, AAB patterns.
- **4–5**: Extend + create new patterns.

### Measurement / Comparison
- **2–3**: Big / Small.
- **3–4**: Tall / Short, More / Less.
- **4–5**: Order by size (seriation).

### Sorting / Data
- **2–3**: Sort by 1 attribute (e.g., color).
- **3–4**: Sort by 2 attributes (e.g., blue circles).
- **4–5**: Simple "which has more?" data comparison.

### Spatial Reasoning
- **2–3**: In / On / Under.
- **3–4**: Here / There, simple fit (puzzles).
- **4–5**: Positional language.

### Logic / Cause-Effect
- **2–3**: Tap → reaction.
- **3–4**: Sequence / order (first, then).
- **4–5**: Simple if-then logic.

## Priority Gaps (Build These Next)
1. **Sort / classify by color, shape, or kind** (not just size).
2. **Pattern depth** (extend and create).
3. **Spatial / positional language** (in / on / under).
4. **More / less and ordering** (sets and seriation).
5. **Measurement** (tall/short, long/short).
