# Skill 02: Learning Design

Use this to evaluate whether gameplay teaches effectively for ages 2-5.

## Learning Loop Standard

Each round should follow:

1. Cue: Simple instruction with one clear objective.
2. Attempt: Immediate child action.
3. Feedback: Specific and supportive response.
4. Reinforcement: Repeat concept in a slightly varied way.

## Adaptive Difficulty Rules

- Start conservative.
- Increase complexity only after consistent success.
- De-escalate quickly after repeated mistakes.
- Prefer reducing choices over repeating the same prompt.

## Hint Ladder Standard

- Wrong 1: verbal cue.
- Wrong 2: visual cue.
- Wrong 3: guided assist.

## Anti-Patterns

- Generic repeated "try again" without new guidance.
- Sudden complexity jumps.
- Multi-step instructions for single-tap activities.
- Rewarding speed over understanding.
