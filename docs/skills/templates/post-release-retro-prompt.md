# Post-Release Retro Prompt Template

Use this after deployment and real-device testing.

---

Run a post-release retrospective for fiona-game.

Focus areas:
- Where children hesitated or repeated mistakes.
- Where instructions were unclear.
- Which interactions felt noisy or overwhelming.
- Parent friction points in settings and controls.
- Device-specific failures (iOS/Android/PWA/offline).

Output:
1. Top 3 wins.
2. Top 3 issues with probable root causes.
3. Prioritized fixes (high/medium/low).
4. One metric to track next release for each fix.
