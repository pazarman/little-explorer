## Systemic Findings

### What the new axes revealed

Every game was re-scored cold against 5 new axes. Three structural patterns emerged that the old rubric was blind to.

#### 1. SEL (axis 4): effectively zero across the library

| Score | Games |
|---|---|
| 1 | Dress-Up (creative affirmation), Pet Care (nurturing mechanic), Story (narrative emotion) |
| 0 | All 21 others |

Pet Care is the closest thing to an SEL game: the pet shows status icons (🫧 🍖 🎾 → 💖) that model basic needs, but emotions are never *named*. No game says "the puppy is sad — let's help!" or "you were so kind!" All top competitors (KAK, Sago Mini, PBS Kids / Daniel Tiger, Noggin) treat SEL as a first-class domain.

#### 2. Active Construction (axis 3): a nearly flat floor

| Score | Games |
|---|---|
| 2 | Dress-Up, Tracing, Music, Paint Studio |
| 1 | All drag-to-target games (counting games, sorting, pizza, etc.) |
| 0 | All binary tap-to-match games (Ocean, Pet Match, Who Says?, Body Match) |

The library is strong on drag mechanics but almost entirely lacks generative output (build, sequence, express freely). Trace, Dress-Up, Music, and Paint are the exceptions.

#### 3. STEM Coverage (axis 5): five games score 0 — all counting

Snow, Dragon Feed, Pasta, Pet Feed, and Dino Eggs all score 0 because they cover the same Number Sense domain that is already **Strong**. Each additional counting game cannibalizes the portfolio rather than expanding it. The library has one game for patterns, one for spatial reasoning, zero for SEL, zero for phonics.

## Backlog (updated for v2 rubric)

### 🔴 High Priority

- [ ] **SEL game** — any mechanic that names an emotion, shows its cause, and has the child respond. Even one game would move the axis from 0 to 2 across the category. Daniel Tiger format (episode + task) or a "feelings sorting" game (match face to situation).
- [ ] **Phonics / literacy game** — letter sound matching, rhyme tap, syllable clap. Fills the single highest-ROI gap per skill 06 and the research literature. Homer and Duolingo ABC patterns are the model.
- [ ] **Memory: Learning Efficacy (axis 1 = 1)** — currently teaches working memory, but the STEM tag claims "Logic/cause-effect" which is inaccurate. Either re-scope as a standalone "Memory" domain game and accept it, or add a logic/sequencing component. The game sits exactly at the pass threshold (18/24).

### 🟡 Medium Priority

- [ ] **Counting games: STEM Coverage (axis 5 = 0)** — Snow, Dragon, Pasta, Pet Feed, Dino all duplicate the Strong number-sense domain. No action needed *now* (they pass) but each new counting game is a STEM Coverage 0 by default. Evaluate before adding more.
- [ ] **Body Match: hint ladder** — wrong taps repeat the instruction but no progressive hint (highlight → wiggle) like other games. Also, small body parts (tongue r=8, knee r=15) are below 44px at typical phone sizes. Fix both to raise score from 19 → 21+.
- [ ] **Ice Cream: adaptivity depth** — pacing scores 1 (backlog from v1). Tier 2/3 logic incomplete. Add dynamic color set + speed ramp to raise from 19 → 21.
- [ ] **Pet Care: SEL upgrade** — this game is one step away from being the library's SEL anchor. Add explicit emotion naming: "The puppy looks sad 😢 — let's help!" before tools appear, and "The puppy is happy now! 💖" on completion. Would raise SEL from 1→2 and score from 19→20.
- [ ] **Pizza: Learning Efficacy** — mixing shapes + colors + counting in one game dilutes the concept per round. Consider either (a) splitting into three focused games or (b) making the mode explicit at the start of each round so the concept is always clear. Would raise score from 20→22.

### 🔵 Low Priority

- [ ] **Bike (Numbers): motor and sensory** — moving balloon targets reduce effective hit zone for the youngest players. Slow balloon speed on tier 0, or add a slight auto-snap. Also reduce tree-scrolling layer brightness for calm mode.
- [ ] **Story: pacing** — no difficulty tiers; every child gets the same story regardless of age band. Add a "harder find task" for tier 2 (find a specific emoji among several similar ones).
- [ ] **Music: hint ladder** — wrong key presses are free (just plays that animal). Add a counter after 2 misses: highlight which animals are still in the sequence.
- [ ] **Hide & Seek: vocabulary confirmation** — success speaks `praise()` but doesn't re-state the relation word. Change to: "Yes! [Buddy] is IN the basket! 🎉" to confirm the spatial term.

## Path to "Premier" Status

To reach the level of Khan Academy Kids or Sago Mini, three product-level changes are needed beyond per-game scores:

1. **Add one SEL game** — closes the single most visible gap vs. every top competitor.
2. **Add one phonics/literacy game** — closes the highest-ROI developmental gap for ages 3–5.
3. **Retire or repurpose one counting game** — five near-identical counting mechanics reduce portfolio diversity. Convert Dino Eggs (the most open-ended of the five — no wrong answers, just tap to hatch) into an open-ended counting + creative game to raise its Active Construction score.
