## 🛡️ Rubric Gaps (Roadmap for "Premier" Status)

To move from "High Quality" to "Premier" (competing with Sago Mini/Khan Academy Kids), the following axes must be added to the rubric and implemented across the app:

### 1. Aesthetic Intent (World Cohesion)
- **Gap:** Current rubric focuses on restraining overstimulation (Sensory Balance) but doesn't mandate stylistic beauty or immersion.
- **Goal:** Every game must feel like it belongs to the same high-polish universe.

### 2. Progression & Persistence (Metagame)
- **Gap:** Rubric scores isolated interactions. It lacks a "Long-term Engagement" metric.
- **Goal:** Scoring how the game remembers progress and adapts the world/rewards over time (e.g., meaningful sticker book evolution).

### 3. Conceptual Scaffolding
- **Gap:** Adaptivity scores "getting easier," but not "teaching better" (e.g., contrasting a mistake with the correct answer).
- **Goal:** Active misconception handling.
