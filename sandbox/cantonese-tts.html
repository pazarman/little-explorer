No Cantonese voice? How to install one

iPhone/iPad: Settings → Accessibility → Spoken Content → Voices → Chinese → Cantonese (Hong Kong) (e.g. “Sinji”). Download it, then reload this page.
Android: Settings → System → Languages & input → Text-to-speech output → choose Google Text-to-Speech → Install voice data → Chinese (Hong Kong) / 粵語. Then reload.
Mac (Safari/Chrome): System Settings → Accessibility → Spoken Content → System Voice → Manage Voices → Chinese (Hong Kong).

The real app uses rate ≈ 0.95, pitch ≈ 1.25 (warm, slightly high for toddlers).
