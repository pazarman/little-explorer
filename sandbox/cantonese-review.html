No Cantonese voice? How to install one

iPhone/iPad: Settings → Accessibility → Spoken Content → Voices → Chinese → Cantonese (Hong Kong) (e.g. “Sinji”), then reload.
Android: Settings → System → Languages & input → Text-to-speech output → Google Text-to-Speech → Install voice data → Chinese (Hong Kong) / 粵語, then reload.
Mac: System Settings → Accessibility → Spoken Content → System Voice → Manage Voices → Chinese (Hong Kong).
